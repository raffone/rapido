9.0 - Grids

Grids usually are defined directly in the scss/sass files using the columns mixin. This is the best solution and keep the html free of classes that one day may be changed requiring more work that editing the stylesheet file and recompiling.

Sometimes is easier to add a class to the html for quick prototiping, in this case there are available several classes just for that.

In this example the columns width is 3 on desktop, 6 on tablet and 12 on smartphone.

9.1 - Show and Hide

There're also available some helper classes:

  • Hide element: Used to hide an element at a breackpoint
    • .palm-hide
    • .lap-hide
    • .portable-hide
    • .lap-and-up-hide
    • .desk-hide
  • Show element: Used to show an element at a breackpoint
    • .palm-show
    • .lap-show
    • .portable-show
    • .lap-and-up-show
    • .desk-show
