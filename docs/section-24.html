24.0 - Functions

For 99% of the framework I used Compass' and Susy's functions but I've made some for covering edge cases.

24.1 - em()

To convert PX to EM using the $base-font-size as font size. 

width: em(20px);

24.2 - neg()

Convert a value to negative.

Example: 

left: neg(10em);

Become: 

left: -10em

24.3 - rhythm_neg()

Create a negative rhythm() value. More info about the rhythm function: Vertical Rhythm.

Example: 

margin-left: rhythm_neg(1);

Become: 

margin-left: -1.42857em;

24.4 - space_neg()

Similar to rhythm_neg() but for Susy's space() function, more info: Susy Reference.
