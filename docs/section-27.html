27 Functions

For 99% of the framework I used Bourbon' and Neat's functions but I've made some for covering edge cases.

27.2 Neg

Convert a value to negative. 

neg(<$value>)
  • $value: The value to convert

27.3 Columns

Similar to span-columns mixin, but returns the math-ready % multiplier. Taken from Susy One. 

columns(<$columns> [, <$context>, <$style>])
  • $columns: The number of columns to span.
  • $context: The Context. (default: $total-columns).
  • $style: Optionally return static lengths for grid calculations. (default: $container-style).
.item { width: columns(3,6); }

27.4 Gutter

The % width of one gutter in any given context. Taken from Susy One. 

gutter([<$context>, <$style>])
  • $context: The Context. (default: $total-columns).
  • $style: Optionally return static lengths for grid calculations. (default: $container-style).
.item { margin-right: gutter(6) + columns(3,6); }

27.5 Space

Total % space taken by Columns, including internal AND external gutters. Taken from Susy One. 

space(<$columns> [, <$context>, <$style>])
  • $columns: The number of Columns to span,
  • $context: The Context. (default: $total-columns).
  • $style: Optionally return static lengths for grid calculations. (default: $container-style).
.item { margin-right: space(3,6); }

27.6 Rhythm

Calculate rhythm units. Vertical Rhythm. 

rhythm(<$lines> [, <$font-size>, <$offset>])
  • $lines: Number of lines
  • $font-size: Context font size (default: $base-font-size)
  • $offset: Offset value (default: 0) 
.item { height: rhythm(4); }
