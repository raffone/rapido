5.0 - Captions

Add captions to images. The position is defined by data-position and the content by data-content. It's possible to add an animation using data-animation with either fade or float.

For positioning are available: top, right, bottom, left, top-left, top-right, bottom-right, bottom-left.

Default styling

Add fade-in animation for the caption

fade

Add fade-in and slide-in animation for the caption

float

 
<figure class="caption" data-position="bottom" data-animation="{$modifiers}" data-content="Caption text.">
  <img src="http://dummyimage.com/300x200/afe600/fff" />
</figure>
